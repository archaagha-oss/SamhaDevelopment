## Broker Model (Corrected)

A **Broker Company** is a registered real estate firm (licensed by RERA).
A **Broker Agent** is an individual employee at that firm who brings leads.

```
Dubai Homes Real Estate  ← Broker Company (has RERA license, trade license)
  ├── Khalid Al Rashidi  ← Broker Agent (has RERA card, brings leads)
  ├── Sara Johnson       ← Broker Agent
  └── Omar Bin Faisal    ← Broker Agent

Commission is always owed to the COMPANY, not the individual agent.
```

### Week 15 — Commission Ledger

**Day 71–72: Commission Records**
- [ ] Commission records already created in Phase 3 (one per deal with broker)
- [ ] Commission status flow:
  ```
  NOT_DUE → PENDING_APPROVAL (auto: SPA signed + Oqood registered)
  PENDING_APPROVAL → APPROVED (ADMIN or FINANCE manual action)
  APPROVED → PAID (upload payment proof → status → PAID)
  ANY → FORFEITED (auto: deal cancelled before Oqood)
  ```
- [ ] `PATCH /api/commissions/:id/approve` — ADMIN/FINANCE role only
- [ ] `PATCH /api/commissions/:id/mark-paid` — requires document upload first
