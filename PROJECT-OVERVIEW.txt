QUICK START (5 MINUTES)
================================================================================

1. Install:
   cd samha-crm && npm install

2. Configure:
   cp apps/api/.env.example apps/api/.env
   cp apps/web/.env.example apps/web/.env

3. Database:
   npm run db:push
   npm run db:seed

4. Run:
   npm run dev

5. Open:
   http://localhost:5173
